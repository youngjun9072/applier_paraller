# 실험 결과 정리

## 관찰

- 시작시간과 종료시간을 확인했을 때, 각 테이블 작업이 **병렬적으로 수행되고 있는 것으로 보임**
  (마스터·슬레이브 모두 여러 테이블의 시작 시각이 거의 동시에 찍힘)
- 그러나 **실질적인 성능 향상은 관찰되지 않음**
  (병렬 처리 구조임에도 전체 복제 시간이 단축되는 효과가 명확하지 않음)

## 다음 액션

- **병목 발생 여부 확인**을 위해 **워커 수를 변경하며 추가 테스트 수행 예정**
- 워커 수 증감에 따른 복제 시간 변화 추이를 측정하여, 실제 병렬화가 효과적으로 동작하는지 / 어느 지점에서 병목이 발생하는지 확인
